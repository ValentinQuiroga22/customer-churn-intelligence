# AI Churn Explanation Engine

The project includes an Explainable AI layer built on top of the churn prediction model.

Instead of providing only a churn probability score, the system identifies the main factors contributing to customer churn risk and generates actionable retention recommendations.

## Example Output

Risk Score: 68.4%

Main Risk Factors:

* Month-to-month contract
* Short customer tenure
* Fiber optic internet service
* No online security service

Recommended Actions:

* Offer discount for a long-term contract
* Promote Online Security package
* Launch onboarding retention campaign
* Review customer satisfaction with Fiber Optic service

## Business Value

This approach transforms raw machine learning predictions into actionable insights that can be used by Customer Success, Marketing, and Retention teams.

The system helps decision-makers understand not only which customers are at risk, but also why they are at risk and what actions can be taken to reduce churn.
